> A GitHub App built with [Probot](https://github.com/probot/probot) that a community bot for tidb.

## Require

- Git >= 2.13.0 (**For husky support**)
- Node >= 10
- MYSQL 5.7
- Docker
- Docker Compose >= 3

## Setup

```sh
# Install dependencies
npm install

# Run with hot reload
npm run build:watch

# Compile and run
npm run build
npm run start
```

## Deploy

```sh
# Docker compose up
docker-compose up -d --build

```
